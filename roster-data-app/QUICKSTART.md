## Getting Started in 5 Minutes

### 1. Open the App
- Open Safari on your iPad
- Go to: `https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/src/`
- Tap "Add to Home Screen" for easy access

### 2. Start Your First Event
1. **Enter your name** as the operator
2. **Enter event name** (e.g., "Basketball Tryouts 2025")
3. **Upload roster file**:
   - Tap "Choose File"
   - Select your CSV roster file
   - Or use the included `TestRoster.csv` for testing
4. **Tap "Start Event"**

### 3. Collect Measurements
1. **Select a person** from the roster grid
2. **Check "Present"** if they're participating
3. **Enter measurements** (each value twice for validation):
   - Height with shoes
   - Height without shoes
   - Reach, Wingspan, Weight
   - Hand length, Hand width
4. **Choose units** (inches/cm, lbs/kg) for each measurement
5. **Add comments** if needed
6. **Tap "Save Measurements"**

### 4. Export Your Data
1. **Tap the export button** (ðŸ“¤) in the header
2. **CSV file automatically saved** to "Manual entry" folder on your iPad
3. **Email app opens** with pre-filled subject/body and automatic attachment
4. **File is automatically attached** (if supported by email client)
5. **Send immediately** or add recipients and send
